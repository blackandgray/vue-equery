# vue-equery
A vue selector plugin

================================

npm：
npm install vue-equery --save

cnpm：
cnpm install vue-equery --save

main.js:

import Vue from 'vue'
import equery from 'vue-equery'

Vue.use(equery)

================================

exm：

vue.$ele('#id')

vue.$ele('.classname')

vue.$ele('tagname')
